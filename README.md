# 点云科技前端工具函数库`@pointcloud/pcloud-utils`

## 基本指令

-   **启动服务**

    ```
    npm start 或 npm run start
    ```

-   **构建项目**

    ```
    npm run build
    ```

-   **构建文档**

    ```
    npm run doc
    ```

-   **构建本地 Docker 镜像**

    ```
    npm run docker:build
    ```

-   **自动化发布**

    ```
    npm run release
    ```

-   **推送 Docker 镜像**

    ```
    npm run docker:push
    ```
